Try Linux!

There are many flavors of Linux out there, each with own strengths and weaknesses. Pick one that you like!

Recommended by Linux Promotion

We recommend a distribution called Fedora. We consider it a good choice for beginners. It has many features that we enjoy:

  • Easy to use.
  • Ships modern, up-to-date software.
  • New versions are released every 6 months.
  • Cares about freedom, doesn't ship any proprietary software except absolutely essential drivers.
  • Doesn't modify the software it ships.

Get Fedora

Other distributions

If you don't like our recommended distribution you can pick any of the other easy to use distributions and try it instead. Here's a list of the most common distributions. Note that we can't cover all the Linux distributions here: there are literally hundreds of them. Visit DistroWatch for even more distributions.

This list is still very incomplete!

Easy to use

These popular distributions are for beginners, of course!

Debian
  • Ships only stable, tried and true software.
  • Very large software repository, contains more than 20,000 packages.
  • No fixed release schedule, new new versions are released when they're ready.
  • Proprietary software needs explicit confirmation to be installed.
  • Software is quite outdated, as it's tested for a long time to ensure stability.
  • Long time between releases.
Get Debian
Trisquel
  • Only ships free software. There is no proprietary software at all in its official repositories.
  • No fixed release schedule, new new versions are released when they're ready.
  • Long time between releases.
Get Trisquel
Mageia
  • Powerful and easy to use control center.
  • Proprietary software needs explicit confirmation to be installed.
  • Long time between releases.
Get Mageia
Ubuntu
  • Easy to use.
  • Large software repository.
  • Explicitly recommends proprietary sofware.
  • Sends user search queries to partners for advertising purposes by default.
Get Ubuntu

Difficult to use

We recommend these for people who are either Linux experts or learn very quickly.

Arch Linux Exactly like Parabola, but doesn't care about freedom.
  • Provides exceptional documentation.
  • Cutting-edge software (software is usually packaged as soon as it's released).
  • No discrete releases, software is updated gradually.
  • No graphical installation and configuration tools, has to be configured manually via command line.
  • Cutting-edge software is not always stable.
  • Proprietary software can be installed by default without explicit confirmation.
Get Arch Linux
Gentoo
  • Software is shipped as source code and compiled on users' computers. This allows the users to leave out features they don't need and to specifically optimize software for their processors.
  • Proprietary software needs explicit confirmation to be installed.
  • No graphical installation and configuration tools, has to be configured manually via command line.
  • Needs a lot of patience, as compilation takes a long time.
Get Gentoo
