Our mission

To promote Linux as much as possible by setting up booths in town and city centers, giving out discs and showing people live Linux systems. We hope to have 55% of home desktop users use Linux.

What is Linux?

Linux is an operating system for computers, just like Windows or Mac OS X.

Why use Linux when I've got Windows?

  • Linux is free of charge. That's right — you get a full operating system complete with applications, all for free!
  • Linux gives you freedom. Linux and most of its applications are free software. You can absolutely legally study how they work, modify and improve them, fix errors, share them with others for free and even sell them. Even if you can't program, you can ask or pay any programmer to do the changes you want for you.
  • Linux will never betray you. Linux will never present you with nasty surprises like forced updates, removing programs and data without your contest or spying on you. Read how proprietary software betrays its users.
  • Linux is secure. Linux is developed with security in mind. Most of the websites, such as Google, Twitter and Facebook, are served to you by Linux. Many militaries use Linux on their servers.
  • Linux is highly customizable. You're not forced to use a graphical user interface you're not comfortable with. There a lot of varieties of user interfaces in Linux, from shiny and hardware-demanding (KDE, GNOME, Enlightenment) to simplistic and lightweight (LXDE, XFCE) and unconventional (i3).
  • Linux is lightweight. Linux is a lot lighter on hardware resources than Windows, enough to revive a very ancient computer but provide as much functionality as Windows does. Most of the non-essential but hardware demanding features such as desktop effects can be toned down or turned off.
  • Linux comes in lots of flavors. The word "Linux" actually refers to a family of operating systems based on the Linux kernel, rather than to a single operating system. This is a natural consequence of Linux being free software: people are not prohibited from making their own Linux based operating systems and they do so. There are many different varieties of Linux, each with own strengths and weaknesses.
  • Linux has powerful developer tools. You can write Linux application in pretty much any programming language: C, C++, Objective-C, C#, VisualBasic.NET, Java, FreePascal (TurboPascal and Delphi compatiable), JavaScript, Perl, Ruby, Python are all included. Linux includes powerful integrated development environments, such as Eclipse, Code::Blocks, CodeLite, MonoDevelop. There are no limited "educational" versions of these tools, they allow you to harness the full power of your computer for free!

Convinced already?

Linux can run on pretty much any computer that isn't locked out of it by manufacturer.

You can boot full Linux desktop from a CD or a pendrive without making any changes to your computer. If you like it, you can install it permanently, either alongside your existing operating system or instead of it.
